Wat is een operatingsystem?

Een operating system zorgt ervoor dat software en hardware met elkaar samenwerkt. Het operatingsystem zorgt er ook voor dat software kan draaien. Het vormt een soort brug tussen hardware en de gebruiker. Als er dus geen besturingssysteem was, zou je een computer veel moeilijker kunnen gebruiken.
